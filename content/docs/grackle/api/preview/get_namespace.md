Gets a namespace.

Request:

```json
{
  "name": "UserObjects"
}
```

Response:

```json
{
  "namespace": {
    "name": "UserObjects",
    "description": "Some description",
    "created_at": 1695826239671432000,
    "updated_at": 1695826239671432000
  }
}
```
